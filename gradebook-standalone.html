Overall grade calculation

Formula reference  (this = this cell; a, b, … = other columns' codes)
Per component — SCALE(n) scale by n% · if(cond, a, b) · clamp(this, lo, hi) · min/max/round(this) · arithmetic on this and codes · a leading + prepends this
Parent fold — DROP(LOWEST[,n]) / DROP(HIGHEST[,n]) · KEEP(a) shields a code from the drop · SUM · MAX · MIN · WAVG(a,b,…) mean weighted by each code's %
Grade (stages, comma-separated) — ROUND() · ROUNDUP / ROUNDDOWN · BOOST(n) lift to a letter boundary within n · BUMP(a->b, …) remap exact values
Missing values — or(a, b) first that exists · orzero(a) = or(a, 0)
Move weight — if(this<50, moveto(a)) hands this component's weight to code a when the test holds (e.g. a failed midterm onto the final)
Plain words work too — less than, at least, at most, move weight to … e.g. If MT is less than 48, move weight to FIN
Course Raw — *** as a code or formula marks the course Raw column · Cap rule — a condition over codes → a maximum grade
