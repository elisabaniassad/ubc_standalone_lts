# UBC standalone learning tools

Single-file, no-server tools an instructor can run by double-clicking —
nothing leaves the machine except the files you export.

| Tool | What it does |
|------|--------------|
| `gradebook-standalone.html` | The Lifeboat gradebook as a converter: import a CSV of marks (plain or Canvas gradebook export), compose grades with per-column formulas/short codes/weights (UBC letters, `+BOOST` with boundary flagging, DNW), read the distribution (histograms, mean/std/As/Fails/DNWs), and **Export for Workday** in the SIS grammar. Browser storage self-clears 2 hours after the last change — Download data is the durable save. |
| `gradebook-standalone-v2.html` | The same converter in the Lifeboat layout: an **Overall grade calculation** table (components, short codes, formulas, weights, the course Raw/Grade formulas, TOTAL flag) sits above the marks grid; the grid slims to marks + In chart / Average / Histogram. |
