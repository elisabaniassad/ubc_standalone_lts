# UBC standalone learning tools

Single-file, no-server tools an instructor can run by double-clicking —
nothing leaves the machine except the files you export.

| Tool | What it does |
|------|--------------|
| `gradebook-standalone.html` | The Lifeboat gradebook as a converter: import a CSV of marks (plain or Canvas gradebook export) and compose grades in an **Overall grade calculation** table above the marks grid — components with short codes, formulas, weights and hierarchy (indent = sub-components), course Raw/Grade formula stages, **grade cap rules**, per-student overrides with a reasons ledger, UBC letters, DNW, distribution charts (filter matches picked out in gold), readable text **scheme files** (structure + exceptions, no marks), and **Export for Workday** as `.xlsx` in the SIS grammar. Browser storage self-clears 2 hours after the last change — Download data is the durable save. |
